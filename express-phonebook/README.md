## Start Expressjs Phonebook API
1. cd expressjs_project_folder 
2. npm install  
3. node bin/www

## API END POINT
http://localhost:3000/api/phonebooks

## Header Parameters
Content-Type  application/json

## Request body to create phonebook entry

{
 "phonebook": {
  "firstname": "aman",
  "lastname": "patial",
  "phonenumber": "9900273410"
  }
}
